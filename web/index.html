Product Performance vs Benchmark
Return 1M / 3M / 6M / 1Y tiap produk dibanding indeks acuan · Gap negatif = sinyal alert underperform
Cara kerja: Update return produk & benchmark di panel ini lalu klik 🔍 Run Detection — prosedur Oracle PROC_PUSH_UNDERPERFORM_ALERTS membandingkan return 3M produk vs benchmark dan membuat alert untuk setiap nasabah pemegang produk yang underperform ≥ threshold yang dikonfigurasi di Admin → Alert Threshold Configuration.
🔴 Gap ≤ −6% = HIGH severity  ·  🟡 Gap ≤ −2% = Medium severity  ·  🟢 Gap ≥ 0% = Outperform
